== Description ==
A really simple and light plugin that adds a bunch more widgets for the SiteOrigin Page Builder. Yes the Page Builder is required as well as the SiteOrigin Widgets Bundle (https://wordpress.org/plugins/so-widgets-bundle/). The idea behind this was to build out some simpler widgets for my specific requirements. A little less bloat. Less features.

Widgets Include:

* Accordion Widget
* Custom Slider Widget
* Icon Box Widget
* Post Slider Widget
* Simple Button Widget
* Tabs Widget

== Installation ==
1) Download the plugin via WordPress.org.
2) Upload the ZIP file through the \"Plugins > Add New > Upload\" screen in your WordPress dashboard.
3) Activate the plugin through the \'Plugins\' menu in WordPress
4) Go to Plugins -> SiteOrigin Widgets and activate whichever ones you want
